1. index.ts
2. utils/utils.ts
3. utils/utils.test.ts

### To run this project:
1. `npm install`
2. `npm build`
3. `npm start {file path}`

##### Example: 
`npm start '.\test case\test_failure.ts'`
`npm start '.\test case\test_success.ts'`
